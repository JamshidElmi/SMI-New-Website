# Shared Mobility Inc. (SMI) Design System Rules

## 1. Brand Foundation
- Primary brand colors:
  - SMI Blue: `#05527E`
  - SMI Green: `#8DC641`
- Supporting palette:
  - Blue 900: `#033955`
  - Blue 800: `#044A70`
  - Blue 600: `#0B6799`
  - Blue 500: `#197BAF`
  - Blue 300: `#6EA6C3`
  - Blue 100: `#E8F3F8`
  - Green 900: `#4E7D20`
  - Green 700: `#6EA529`
  - Green 300: `#B9DE83`
  - Green 100: `#F1F8E8`

## 2. Typography
### Font Family
- Use **Dosis** throughout the website.

### Headings
- Uppercase headings.
- Strong letter spacing.
- Heavy font weights (700–800).
- Primary heading color: SMI Blue.

### Type Scale
- H1: 46–86px
- H2: 42px
- H3: 26px
- Body: 18px
- Lead text: 24px
- Caption: 14px uppercase

## 3. Layout
- Max content width: **1180px**
- Use generous white space.
- Sections should feel open and breathable.
- Content containers use subtle shadows and light borders.

## 4. Corner Radius
Keep roundness soft but restrained.

- Small radius: 8px
- Medium radius: 14px
- Large radius: 24px
- Section radius: 28–30px
- Buttons: Pill radius (999px)

## 5. Shadows
### Primary Shadow
`0 18px 45px rgba(5,82,126,.14)`

### Secondary Shadow
`0 8px 22px rgba(5,82,126,.10)`

Use shadows lightly to create depth.

## 6. Buttons

### Primary Button
- Background: SMI Blue
- Text: White
- Border: 2px solid SMI Green
- Shape: Pill

### Primary Hover
- Background: SMI Green
- Text: Blue 900
- Slight lift animation

### Secondary Green Button
- Background: SMI Green
- Text: Blue 900

### Secondary Hover
- Background: SMI Blue
- Text: White

### Outline Button
- Transparent background
- Green border
- Blue text

### Ghost Button
- Blue-100 background
- Green border
- Blue text

### Motion
- Hover translateY(-2px)
- Enhanced shadow on hover

## 7. Forms
### Inputs
- 2px border
- Rounded corners (14px)
- White background

### Focus State
- Blue border
- Green focus ring

## 8. Cards
- White background
- Light border
- Soft shadow
- Rounded corners
- Image-led layouts encouraged

## 9. Navigation
- Clean, minimal navigation
- Uppercase labels
- Wide spacing
- Blue text on light backgrounds

## 10. Content Style

### Visual Tone
- Professional
- Community-oriented
- Accessible
- Human-centered
- Transportation-focused

### Content Structure
- Use clear section hierarchy.
- Lead with impactful headlines.
- Support with imagery from programs, communities, bicycles, and mobility projects.

## 11. Neuro Web Design Principles

### Attention
- Use green as a highlight color.
- Maintain strong visual contrast.
- Use imagery with people whenever possible.

### Trust
- Show partners and supporters.
- Use real project photography.
- Display measurable impact.

### Clarity
- Limit visual clutter.
- Keep layouts simple.
- Make calls-to-action obvious.

### Scannability
- Large section headings.
- Short paragraphs.
- Clear card layouts.

## 12. Homepage Direction

Based on project notes:

### Primary Inspiration
- Toole Design website structure and layout.

### Secondary Inspiration
- Pedal Movement video hero treatment.

### Project Section
- Follow the Toole Design project-grid approach.
- Large visual project cards.
- Image-first presentation.
- Strong project titles and CTA buttons.

## 13. Design Personality

The website should feel:

- Modern
- Credible
- Community-driven
- Mission-focused
- Optimistic
- Professional
- Human-centered
- Transportation and mobility focused

## 14. Do / Don't

### Do
- Use SMI Blue as the dominant brand color.
- Use SMI Green for emphasis and interaction.
- Keep corners softly rounded.
- Use real photography.
- Create clear visual hierarchy.
- Use generous spacing.

### Don't
- Overuse shadows.
- Use sharp corners.
- Create cluttered layouts.
- Use generic stock imagery when real project imagery exists.
- Overcomplicate navigation.
